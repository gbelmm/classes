Exercise I: Event Data Modeling

  • Find someone in the class to pair with you.
  • Pick an example of an event in your website or application that you would like to record. Examples: signups, purchases, clicks, shares, content creations, comments.
  • Create a data model for that event. Write out a list of properties you would record about that event.
  • Think of two ways you could use this event data to improve your app.
  • We will select a few examples from the class to talk through
