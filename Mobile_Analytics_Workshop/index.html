Instructions for Integration Exercise

  1. Create a Keen IO Account
  2. Install the iOS Client using the iOS Usage Guide. If you don't have an iOS dev environment, you can try this exercise by sending events using cURL.
  3. Send at least one event! It'll be more fun if you send more.
    • Remember, the name of the event collection should be an action like "Search" or "Submit".
    • The event can have any number of properties you want, as long as it is in a dictionary format.
  4. Login to Keen.io and check that the event came in on the Project Settings page (check last 10 events)
  5. Use the API workbench to count the number of events sent
  6. Use the API workbench to apply a filter to the event count
  7. Run an extraction to see what your event data looks like in Excel

If you have questions, ask Dan or Michelle or use users.keen.io chat room!
